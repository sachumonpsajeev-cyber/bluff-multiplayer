# 🃏 Real-Time Multiplayer Bluff (Cheat)
A full-stack, real-time multiplayer card game built to demonstrate WebSockets and State Management.

## 🚀 Features
- **Real-Time Gameplay:** Powered by Socket.io for instant card plays and turn updates.
- **Dynamic Lobby:** Join with a custom username and wait for friends.
- **Bluff Detection:** Logic-based "Challenge" system to catch liars.
- **Responsive UI:** Clean, casino-themed interface built with CSS Flexbox.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** Node.js, Express
- **Real-Time:** Socket.io

## 📖 How to Run Locally
1. Clone this repo.
2. Run `npm install`.
3. Start the server with `node server.js`.
4. Open `http://localhost:3000` in multiple tabs.
